# سجلات الأمان — فلاش بيور

هذا الفولدر بيحتوي على سجل يومي لكل ما يتعلق بأمان المنصة.

## هيكل الملفات
```
security-logs/
  SECURITY-LOG-YYYY-MM-DD.md   ← سجل يومي
  README.md                     ← هذا الملف
```

## محتوى كل ملف يومي
- الحالة العامة للنظام
- الثغرات المكتشفة وحالة إصلاحها
- محاولات الدخول المشبوهة
- IPs المحجوبة
- ملاحظات مهمة

## قواعد مهمة
- الملف بيُنشأ كل يوم حتى لو ما فيش أي نشاط مشبوه
- مفيش حد يقدر يعدل أو يمسح ملف موجود
- كل ثغرة لازم تُكتب فيها: الملف، المشكلة، الخطر، الإصلاح، والحالة
